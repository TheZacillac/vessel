# Vessel

> **Status: Pre-development** — This project does not yet contain a model. Check back later.

Vessel is the home for the trained domain expert LLM within the [Arcanum Suite](https://github.com/TheZacillac). Once training is complete, this repository will house the fine-tuned model weights, configuration, and inference tooling.

## Dataset

Training data is generated by [Oracle](https://github.com/TheZacillac/oracle), the Arcanum Suite's synthetic dataset generator covering 423 topics across 26 categories of domain name expertise.
